# Networking Basics (Cybersecurity Perspective)

## What is Networking?
Networking is the communication between devices using protocols.

## Why it matters in Cybersecurity
- Attackers exploit network misconfigurations
- SOC analysts analyze network traffic
- Firewalls and IDS work at network level

## Key Concepts
- IP Address
- TCP vs UDP
- Ports
- DNS
- HTTP vs HTTPS
